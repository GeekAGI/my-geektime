### my geektime
This is a tool to obtain the geektime video or articles with you geektime account

---
#### Install

```shell
go install github.com/zkep/mygeektime
```

Commond help:
```shell
mygeektime -help
```

Commond help output:
```shell
My GeekTime CLI 0.0.1

Available commands:

   server   This is http server 
   cli      This is Command 

Flags:

  -help
        Get help on the 'mygeektime' command.

```

#### Http Server

```shell
mygeektime server
```


#### Command

```shell
mygeektime cli 
```
