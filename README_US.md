#### Install

#### install with docker compose

```shell
git clone https://github.com/zkep/my-geektime.git

cd my-geektime/docker

# Apple M1 , M2 modify docker-compose.yml 24 line use zkep/my-geektime:mac-m

docker-compose up -d
```
